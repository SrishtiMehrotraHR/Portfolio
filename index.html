Recruitment funnel analytics

The same recruitment dataset, analyzed two ways — SQL for the query layer, Excel for the dashboard layer most TA teams actually work in day to day.

-- Which channels convert best?
SELECT source,
  COUNT(*) AS total,
  SUM(CASE WHEN status='Hired' THEN 1 ELSE 0 END) AS hired
FROM candidates
GROUP BY source
ORDER BY hired DESC;
